# EventScan - Помощник для поиска IT-мероприятий

## Краткое описание
EventScan - это Telegram-бот, который помогает находить актуальные IT-мероприятия, конференции, митапы и хакатоны. Бот предоставляет персонализированные рекомендации на основе вашей должности и позволяет записываться на мероприятия. Для руководителей компаний доступна возможность записи своих сотрудников на мероприятия.

## Основные функции

### Для всех пользователей:
- Поиск актуальных IT-мероприятий по ключевым словам
- Поиск по архиву прошедших мероприятий с подробной информацией
- Личный календарь записанных мероприятий
- Просмотр и изменение личных данных

### Для руководителей компаний:
- Запись сотрудников своей компании на мероприятия
- Персонализированные рекомендации мероприятий для команды
- Управление мероприятиями для всей компании

## Начало работы

1. **Нажмите кнопку "Start"** или отправьте команду `/start`
2. **Пройдите регистрацию**, следуя инструкциям бота:
   - Введите ваше ФИО
   - Укажите электронную почту
   - Название вашей компании
   - Вашу должность
3. После регистрации бот автоматически подберет для вас персональные IT-мероприятия.

## Основное меню

После регистрации вы увидите главное меню с кнопками:

### Поиск по архиву
Позволяет искать информацию о прошедших мероприятиях. Просто введите ключевые слова (например, "машинное обучение", "кибербезопасность"), и бот найдет релевантные мероприятия из архива.

### Найти мероприятия
Предлагает два варианта поиска:
- **По моей должности** - автоматический поиск мероприятий, релевантных вашей должности
- **Ввести свой запрос** - поиск по любым ключевым словам, которые вы укажете

### Записать сотрудника (только для руководителей)
Доступно только для пользователей с управляющими должностями (директор, руководитель, начальник отдела и т.д.). Позволяет:
1. Выбрать сотрудника из списка вашей компании
2. Ввести название мероприятия
3. Сотрудник будет записан на мероприятие (мероприятие появится в его календаре)

### Мой календарь
Отображает все мероприятия, на которые вы записаны. Если календарь не пустой, доступна кнопка "Очистить календарь" для удаления всех записей.

### Мои данные
Показывает вашу регистрационную информацию:
- ФИО
- Электронная почта
- Название компании
- Должность
- Уникальный ID
- Дата регистрации
- Username в Telegram

## Особенности для руководителей

Если ваша должность содержит слова: "директор", "руководитель", "начальник", "управляющий", "заместитель директора", "CEO", "владелец" и т.п., бот распознает вас как руководителя и предоставляет дополнительные возможности:

- В главном меню появляется кнопка "Записать сотрудника"
- При первом запуске вы увидите специальное уведомление "Вы являетесь руководителем компании!"
- Вы можете просматривать список всех сотрудников вашей компании (кроме других руководителей)
- Для каждого сотрудника вы можете указать мероприятие, на которое хотите его записать

Важно: бот автоматически определяет сотрудников вашей компании по названию организации, указанному при регистрации.

## Примеры использования

### Пример 1: Поиск мероприятий для Data Scientist
1. Нажмите "Найти мероприятия"
2. Выберите "По моей должности"
3. Бот найдет мероприятия, основываясь на вашей должности
4. Выберите интересующие вас мероприятия

### Пример 2: Руководитель записывает сотрудника
1. Нажмите "Записать сотрудника"
2. Выберите сотрудника из списка (например, "Иван Петров (Frontend-разработчик)")
3. Введите название мероприятия (например, "Frontend Conference 2025")
4. Сотрудник получит запись в свой календарь (бот отправит уведомление)

### Пример 3: Поиск по архиву
1. Нажмите "Поиск по архиву"
2. Введите запрос "IT конференция СПБ"
3. Бот покажет все прошедшие IT конференции в Санкт-Петербурге
4. Вы увидите подробную информацию: даты, место, спикеров, количество участников

## Дополнительные команды

- `/start` - начать работу с ботом или вернуться в главное меню
- `/mydata` - просмотреть ваши регистрационные данные
- `/reregister` - повторная регистрация (удаление старых данных и создание новых)

## Часто задаваемые вопросы

**Вопрос 1: Как бот определяет, что я руководитель?**
Ответ: Бот анализирует вашу должность при регистрации. Если она содержит ключевые слова руководящих позиций, вы получаете доступ к функциям управления.

**Вопрос 2: Могу ли я увидеть календарь своих сотрудников?**
Ответ: Нет, вы можете видеть только свой календарь. Но вы можете записать сотрудника на мероприятие, и оно появится в его календаре.

**Вопрос 3: Что происходит при повторной регистрации?**
Ответ: Все ваши данные (включая календарь) будут удалены, и вам нужно будет пройти регистрацию заново.
